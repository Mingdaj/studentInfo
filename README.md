# studentInfo
Java语言，SSM框架，学生信息管理系统（包含选课）

针对学生、老师和管理员3个角色进行需求分析，运用面向对象的设计方式开发了本系统。

技术栈：
Spring + SpringMVC + mybatis +JSP

搭建环境：
jdk8 + mysql5.7 + maven3.6.1
